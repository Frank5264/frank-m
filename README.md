## To Start
[![Deploy](https://www.herokucdn.com/deploy/button.svg)](https://heroku.com/deploy?)
-`edit config.yml`
install requirements.txt - `pip install -r requirements.txt`
- `python -m mills` - if Windows
- `sudo bash start.sh`- if linux
- `python3 -m mills`
